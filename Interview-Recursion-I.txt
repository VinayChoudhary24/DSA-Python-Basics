## Example with BASE CASE 
def facto_rec(n):
    if n == 0:
        return 1
    return n * facto_rec(n - 1)

n = int(input())
facto_rec(n)

Here, we add the BASE CASE which tells function to stop going deeper if n == 0 and return.


## IMP:: Recursion and PMI(Principle of Mathematical Induction). 
                             ## Approach to Recursion Problem
                                1. we need a BASE CASE.
                                2. what is the Induction Assumption.
                                3. What we need to proof.


# Example with sum of natural numbers

def sum_of_natural_num(n):
    # BASE CASSE
    if n == 0:
        return 0

    # Induction Assumption
    small_output = sum_of_natural_num(n - 1)

    # Task to proof
    nat_sum_output = n + small_output
    return nat_sum_output

n = int(input())
print(sum_of_natural_num(n))

15


###Qsuestion
Find power of a number

Write a program to find x to the power n (i.e. x^n). Take x and n from the user. You need to print the answer.

Note : For this question, you can assume that 0 raised to the power of 0 is 1

Answer:
def power(x, n):
    # BASE CASE
    if n == 0:
        return 1   # by definition: x^0 = 1
    
    # INDUCTION ASSUMPTION
    small_ans = power(x, n - 1)
    
    # INDUCTION STEP (TASK TO PROVE)
    final_ans = x * small_ans
    return final_ans


# Taking input
a, b = input().split()
a = int(a)
b = int(b)
print(power(a, b))

###Question
Find the Nth Fibonacci Number

Answer:
def fib(n):
    # Base Case
    if n == 1 or n == 2:
        return 1

    # Induction Assumption
    fib_n_1 = fib(n - 1) # Last Term
    fib_n_2 = fib(n - 2) # Second Last term

    # Task
    fib_output = fib_n_1 + fib_n_2

    return fib_output

n = int(input())
fib(n)

###Question 
Check if List is Sorted or Not

Answer:
def isSortedOptimized(a, si):
    length = len(a)
    if si == length - 1 or si == length:
        return True
    if a[si] > a[si + 1]:
        return False

    isSmallerSorted = isSortedOptimized(a, si + 1)
    return isSmallerSorted

print(isSortedOptimized([1, 2, 3, 4, 5, 6], 0))

###Question
Sum of array (Recursive)

Given an array of length N, you need to find and return the sum of all elements of the array.

Do this recursively.

Answer:
def sumArrayOptimized(arr, si):
    length = len(arr)

    # Base case
    if si == length:
        return 0

    # Recursive call
    small_sum = sumArrayOptimized(arr, si + 1)

    # Task
    return arr[si] + small_sum


print(sumArrayOptimized([1, 2, 3, 4, 5], 0))

15


###Question
Check Number Recursively

Given an array of length N and an integer x, you need to find if x is present in the array or not. 
Return true or false.

Do this recursively.

Answer:
def checkNumber(arr, x, si):
    # Base Case → reached end of list
    if si == len(arr):
        return False
    
    # If current element matches
    if arr[si] == x:
        return True

    # Recursive call for next index
    return checkNumber(arr, x, si + 1)


arr = list(map(int, input().split()))
x = int(input())

if checkNumber(arr, x, 0):
    print("true")
else:
    print("false")

###Question
Find First Index

Given an array of length N and an integer x, you need to find and return the first index of integer x present 
in the array. Return -1 if it is not present in the array.

First index means, the index of first occurrence of x in the input array.

Do this recursively. Indexing in the array starts from 0.

Answer:
def firstIndex(arr, x, si):
    # Base Case: if we reach end of array
    if si == len(arr):
        return -1

    # If the element is found at current index, return it
    if arr[si] == x:
        return si

    # Recursive Call
    return firstIndex(arr, x, si + 1)

fi_arr = list(map(int, input().split()))
fi_x = int(input())
print(firstIndex(fi_arr, fi_x, 0))

###Question
Find the Last Index

Given an array of length N and an integer x, you need to find and return the last index of integer x present 
in the array. Return -1 if it is not present in the array.

Last index means - if x is present multiple times in the array, return the index at which x comes last in the array.

You should start traversing your array from 0, not from (N - 1).

Do this recursively. Indexing in the array starts from 0.

Answer:
def lastIndex(arr, x, si):
    # Base Case: if we reach end of array
    if si == len(arr):
        return -1

    # Recursive Call
    smaller_list_output = lastIndex(arr, x, si + 1)

    if smaller_list_output != -1:
        return smaller_list_output
    else:
        if arr[si] == x:
            return si
        else:
            return -1

la_arr = list(map(int, input().split()))
la_x = int(input())
print(lastIndex(la_arr, la_x, 0))
